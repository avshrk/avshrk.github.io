Elixir

Create a map. 

 
        
          colors = %{ color: "red" }
          cars = %{  "sedan" => "toyota"}

Access map element 

 
        
          colors[:color] 
          cars["sedan"]

Properties of keyword list 

 
        
        keys must be atoms
        keys are ordered
        keys can be duplicated ( utilized in Ecto / maps you can not have duplicate keys)
        Brackets can be omitted if it is the last argument to function

Update a map 

 
        
        colors 
        |> Map.put(:first, "green")

        %{colors | first: "green"}

Add new property to a map 

 
        
        colors
        |> Map.put(:new_prop, new_value)

Create a keyword list

Access the first lement of a keyword list 

 
        
        colors |> hd
        colors |> List.first
        [frst | rest ] = colors

Run mix within iex 

 
        
        iex -S mix
