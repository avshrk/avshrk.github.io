I am a back-end engineer with a solid background in computer science fundamentals. I have multiple years of experience building software with proficiency in Ruby/Rails, Node/JavaScript, RESTful APIs and containerized deployment.

I created CushionDB, an open-source, data management framework for building offline-first Progressive Web Applications.

CushionDB is the only data management framework that enables data synchronizations independent of web app's status. It will sync data bidirectionally even if the app is closed.

CushionDB is built using Node.js and the latest Progressive Web Application tools. It is comprised of two components: client side database and server side docker containers, Cushion Server and CouchDB.
